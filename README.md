# 🚀 CsharpDotNetDemo

A Proof of Concept (POC) showcasing modern features of **C# 14** and **.NET 10** using a Minimal API approach.

---

## 📌 Objective

This project demonstrates how modern C# and .NET features help:
- Reduce boilerplate code
- Improve readability and maintainability
- Enhance performance
- Simplify dependency injection and API design

---

## 🧱 Tech Stack

- .NET 10 (Preview)
- C# 14 (Preview)
- Minimal APIs
- System.Text.Json

---

## ✨ Features Covered

### 🔹 C# 14 Features

#### 1. Primary Constructors
- Defined directly in class signature
- Reduces constructor boilerplate

#### 2. Pattern Matching (Advanced)
- Property patterns
- Type patterns
- List patterns
- Guard clauses

#### 3. With Expressions
- Immutable object updates using records

#### 4. Collection Expressions
- Simplified array initialization

#### 5. Spread Operator (`..`)
- Merge collections cleanly

#### 6. Raw String Literals
- Cleaner JSON/text without escaping

---

### 🔹 .NET Features

#### 1. Minimal APIs
- Lightweight API design without controllers

#### 2. Keyed Dependency Injection
- Resolve services using keys instead of factories

#### 3. System.Text.Json
- High-performance JSON serialization

#### 4. Span (Performance Optimization)
- Stack-based memory usage for high-performance scenarios

#### 5. Parallel Processing (PLINQ)
- Efficient parallel data processing
